## Setup

Be sure you have `poetry`: <https://python-poetry.org>
Then setup this folder.

```sh
python -m venv .venv
source .venv/bin/activate
pip install -r ./requirements.in
pip freeze > ./requirements.txt
```

## Running

```sh
# For each terminal.
source .venv/bin/activate
```
